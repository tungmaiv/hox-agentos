## Adapter detection priority

In AdapterRegistry.detect_adapter(source):
1. If source starts with `claude-market://` → ClaudeMarketAdapter
2. If source matches GitHub URL pattern (github.com/*/):
   - If URL ends with .md → SkillRepoAdapter (direct SKILL.md import)
   - Else → GitHubAdapter (repo browse)
3. If source is HTTP/HTTPS URL (not GitHub) → SkillRepoAdapter (agentskills-index.json or direct URL)
4. Default: raise ValueError("No adapter found for source")

## GitHub API pattern

GitHubAdapter uses the GitHub API (no auth for public repos):
- List files: `GET https://api.github.com/repos/{owner}/{repo}/git/trees/HEAD?recursive=1`
- Filter to files ending in `SKILL.md` or `skill.yaml` or `skill.yml`
- For each skill file: fetch raw content from `https://raw.githubusercontent.com/{owner}/{repo}/HEAD/{path}`
- Pass raw content to existing `SkillImporter.parse_skill_md()` or `import_from_claude_code_yaml()`

ClaudeMarketAdapter: MVP stub — `can_handle` returns True for `claude-market://` prefix, `fetch_and_normalize` raises `NotImplementedError("Claude Market adapter not yet implemented")`. This prevents errors when source detection works but the actual API isn't wired.

## Security gate (24-05 dependency note)

`UnifiedImportService` must call `scan_skill_with_fallback()` from `backend/security/scan_client.py`. Plan 24-05 creates this module. To avoid a hard dependency between 24-04 and 24-05 (both in wave 2), UnifiedImportService should import it with a try/except fallback:
```python

    from security.scan_client import scan_skill_with_fallback
    _HAS_SCANNER = True
except ImportError:

```
If scanner not available, log a warning and proceed (security gate deferred). This allows 24-04 to execute independently of 24-05 in wave 2.

    ```python
    """UnifiedImportService: route skill imports through adapters with security gate."""
    from uuid import UUID
    from sqlalchemy.ext.asyncio import AsyncSession
    import structlog
    from registry.models import RegistryEntry
    from registry.service import UnifiedRegistryService
    from registry.handlers.skill_handler import SkillHandler
    from skills.adapters.registry import AdapterRegistry
    from skills.adapters.base import NormalizedSkill
    from core.schemas.registry import RegistryEntryCreate

    # Conditional import of security scanner (24-05 may not exist yet)
    try:
        from security.scan_client import scan_skill_with_fallback
        _HAS_SCANNER = True
    except ImportError:
        _HAS_SCANNER = False

    logger = structlog.get_logger(__name__)

    class UnifiedImportService:
        def __init__(self) -> None:
            self._adapter_registry = AdapterRegistry()
            self._registry_service = UnifiedRegistryService()

        async def import_skill(
            self,
            source: str,
            session: AsyncSession,
            owner_id: UUID,
        ) -> RegistryEntry:
            # 1. Detect adapter
            adapter = self._adapter_registry.detect_adapter(source)
            logger.info("skill_import_start", source=source, adapter=type(adapter).__name__)

            # 2. Validate source
            validation = await adapter.validate_source(source)
            if not validation.get("valid", True):
                raise ValueError(f"Invalid skill source: {validation.get('reason', 'unknown')}")

            # 3. Fetch and normalize
            normalized: NormalizedSkill = await adapter.fetch_and_normalize(source)

            # 4. Security gate
            scan_result = {}
            if _HAS_SCANNER:
                skill_data = {
                    "name": normalized.name,
                    "scripts": normalized.instruction_markdown or "",
                    "procedure_json": normalized.procedure_json,
                }
                scan_result = await scan_skill_with_fallback(skill_data, None)
            else:
                logger.warning("security_scan_skipped", reason="scan_client not available")

            # 5. Create registry entry
            config = {
                "instruction_markdown": normalized.instruction_markdown,
                "procedure_json": normalized.procedure_json,
                "allowed_tools": normalized.allowed_tools,
                "tags": normalized.tags,
                "category": normalized.category,
                "source_url": normalized.source_url or source,
                "source_type": normalized.source_type,
                "author": normalized.author,
                "license": normalized.license,
                "version": normalized.version,
                "security_score": scan_result.get("score"),
                "security_report": scan_result,
                "scan_engine": scan_result.get("scan_engine", "none"),
            }
            entry_data = RegistryEntryCreate(
                type="skill",
                name=normalized.name,
                display_name=normalized.name,
                description=normalized.description,
                config=config,
                status="draft",
            )
            entry = await self._registry_service.create_entry(session, entry_data, owner_id=owner_id)
            logger.info("skill_import_complete", name=normalized.name, entry_id=str(entry.id))
            return entry
    ```
