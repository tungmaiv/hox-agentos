If all attempts fail, return `null` and let auth.ts fall back to local auth (do NOT throw — throwing causes the "Server error — Configuration"). Use a local `fetchWithRetry` function:
    ```typescript
    async function fetchWithRetry(url: string, maxAttempts = 3): Promise<Response | null> {
      let delay = 500;
      for (let attempt = 0; attempt < maxAttempts; attempt++) {
        try {
          const res = await fetch(url, { cache: "no-store" });
          if (res.ok) return res;
        } catch {}
        if (attempt < maxAttempts - 1) await new Promise(r => setTimeout(r, delay));
        delay *= 2;
      }
      return null;
    }
    ```
