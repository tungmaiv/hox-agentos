**Goal:** Add three features to the user `/skills` page: a "Promoted" curated section at the top, an export download button on skill cards, and a "Shared with me" section below the main grid.

**Tech Stack:** React 18, Next.js 15 App Router, TypeScript strict, Tailwind CSS

### Task 3: Export download button on user skill cards

```typescript
const handleExport = (skill: SkillItem) => {
  fetch(`/api/skills/${skill.id}/export`)
    .then((res) => {
      if (!res.ok) return;
      const disposition = res.headers.get("Content-Disposition") ?? "";
      const filenameMatch = disposition.match(/filename="?([^"]+)"?/);
      const filename = filenameMatch?.[1] ?? `${skill.name}-${skill.version}.zip`;
      return res.blob().then((blob) => {
        const url = URL.createObjectURL(blob);
        const a = document.createElement("a");
        a.href = url;
        a.download = filename;
        document.body.appendChild(a);
        a.click();
        document.body.removeChild(a);
        URL.revokeObjectURL(url);
      });
    })
    .catch(() => {
      // Silent failure ŌĆö export is a convenience feature
    });
};
```
