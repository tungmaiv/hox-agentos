<!-- API contracts from Plan 01 that frontend will consume -->

GET /api/admin/sso/health (requires admin JWT):
```json
{
  "overall": "healthy" | "degraded" | "unhealthy",
  "categories": [
    {"name": "certificate", "status": "green" | "yellow" | "red", "detail": "Valid, expires in 340 days"},
    {"name": "config", "status": "green", "detail": "All required fields present"},
    {"name": "connectivity", "status": "green", "detail": "JWKS reachable (145ms)"},
    {"name": "performance", "status": "green", "detail": "JWKS latency 145ms"}
  ],
  "circuit_breaker": {"state": "closed", "failure_count": 0, "last_failure_time": null, "thresholds": {"failure_threshold": 5, "recovery_timeout_seconds": 60, "half_open_max_calls": 1}},
  "checked_at": "2026-03-15T10:00:00Z"
}
```

PUT /api/admin/sso/circuit-breaker/config (requires admin JWT):
Request body:
```json
{"failure_threshold": 5, "recovery_timeout_seconds": 60, "half_open_max_calls": 1}
```
Response: 200 with updated thresholds

POST /api/admin/sso/circuit-breaker/reset (requires admin JWT):
Response: 200 (resets circuit breaker to CLOSED)

GET /api/auth/config (public, no JWT):
```json
{
  "auth": "local+keycloak",
  "sso_enabled": true,
  "sso_available": true
}
```
Note: sso_available = false when circuit breaker is open (even if sso_enabled = true)

GET /api/admin/notifications?unread_only=true (requires admin JWT):
```json
[
  {"id": "uuid", "category": "sso_health", "severity": "critical", "title": "SSO Down", "message": "...", "is_read": false, "created_at": "...", "metadata_json": null}
]
```

GET /api/admin/notifications/count (requires admin JWT):
```json
{"total": 5, "unread": 2}
```

POST /api/admin/notifications/{id}/read -> 200
POST /api/admin/notifications/read-all -> 200
