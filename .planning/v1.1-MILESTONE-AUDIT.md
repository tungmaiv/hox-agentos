### Critical Integration Gaps Found

None — all 3 findings below are tech debt (non-blocking):

**Finding 1 — Tech Debt: `blitz_llm_calls_total` never incremented (OBSV-01)**
`core/metrics.py` defines the counter with a comment "instrumented in core/config.py get_llm() wrapper" but `get_llm()` returns a bare `ChatOpenAI` with no instrumentation. Metric always reads 0. Fix: wrap `ChatOpenAI` call or use LiteLLM callback. Impact: LLM usage visible in logs and LiteLLM dashboard (`litellm_spend_metric`) but not in Blitz's own custom Prometheus metrics.

**Finding 2 — Tech Debt: `patch_tool_status` missing cache invalidation (EXTD-03, EXTD-05)**
`admin_tools.py` `patch_tool_status()` and `activate_tool_version()` do not call `invalidate_tool_cache()` after changing tool status. Newly-disabled tools remain callable for up to 60 seconds. Fix: add `invalidate_tool_cache()` call in both functions.

**Finding 3 — Documentation Bug: `list_templates` docstring (WKFL-09)**
`workflows.py` `list_templates` docstring says "no JWT required — templates are public read" but `Depends(get_user_db)` requires JWT. Unauthenticated clients get 401. Fix: update docstring to reflect reality, or change dependency to `Depends(get_db)` if public access is intended.
