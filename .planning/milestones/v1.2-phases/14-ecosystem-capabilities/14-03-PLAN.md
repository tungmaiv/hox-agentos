2. Create `backend/skill_repos/schemas.py`:
   ```python
   class RepoCreate(BaseModel):
       url: str  # Base URL of the repository (serves agentskills-index.json)

   class RepoInfo(BaseModel):
       id: str
       name: str
       url: str
       description: str | None
       is_active: bool
       last_synced_at: str | None  # ISO datetime
       skill_count: int

   class SkillBrowseItem(BaseModel):
       name: str
       description: str | None
       version: str | None
       repository_name: str
       repository_id: str
       metadata: dict[str, Any] | None  # author, license, etc.

   class ImportRequest(BaseModel):
       repository_id: str  # UUID
       skill_name: str

   class ImportResponse(BaseModel):
       skill_id: str
       name: str
       status: str  # "pending_review"
       security_score: int
       security_recommendation: str

   class IndexSchema(BaseModel):
       """Validates the agentskills-index.json format."""
       repository: dict[str, Any]  # name, description, url, version
       skills: list[dict[str, Any]]
   ```

3. Create `backend/skill_repos/service.py` — repo CRUD functions only (browse/import in Task 1b):
   - `async def fetch_index(url: str) -> dict[str, Any]:`
     - Fetch `{url.rstrip('/')}/agentskills-index.json` via httpx (30s timeout)
     - Parse JSON, validate against IndexSchema
     - Return parsed index dict
     - Raise ValueError on invalid structure

   - `async def add_repo(url: str, session: AsyncSession) -> RepoInfo:`
     - Fetch and validate index
     - Check name uniqueness against existing repos
     - Create SkillRepository row: name from index, url, description from index, cached_index=fetched index, last_synced_at=now
     - Return RepoInfo

   - `async def remove_repo(repo_id: UUID, session: AsyncSession) -> None:`
     - Delete SkillRepository row (imported skills in skill_definitions remain)

   - `async def sync_repo(repo_id: UUID, session: AsyncSession) -> RepoInfo:`
     - Load repo, re-fetch index from repo.url
     - Update cached_index and last_synced_at
     - Return updated RepoInfo

   - `async def list_repos(session: AsyncSession) -> list[RepoInfo]:`
     - Query all SkillRepository rows, return as RepoInfo list with skill_count from cached_index

   admin_router = APIRouter(prefix="/api/admin/skill-repos", tags=["admin-skill-repos"])
