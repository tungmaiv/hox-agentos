**Expected:** WorkflowRun status is `failed`; `result_json` contains "Keycloak role fetch failed"; no message delivered.
**Why human:** Requires live Keycloak admin console access to modify client roles.

**Expected:** The node output contains human-readable email summary text (not raw JSON, not stub text), and the text is forwarded to the channel output node.
**Why human:** Requires live LLM (LiteLLM proxy) and real email credentials in the credential vault.
