UniqueConstraint on (artifact_type, artifact_id, role). Index on (artifact_type, artifact_id) for lookup.

    5. **user_artifact_permission.py**: UserArtifactPermission with id (UUID PK), artifact_type (String), artifact_id (UUID, no FK), user_id (UUID, no FK -- Keycloak), allowed (Boolean, default True), status (String, default "active" -- supports staged model), created_at. UniqueConstraint on (artifact_type, artifact_id, user_id). Index on (artifact_type, artifact_id). This supports per-user overrides beyond role-level permissions.

    6. **role_permission.py**: RolePermission with id, role (String), permission (String), created_at. UniqueConstraint on (role, permission). Index on role.

    7. **Evolve mcp_server.py**: Add 4 new columns -- version (Text, nullable), display_name (Text, nullable), status (Text, server_default="active"), last_seen_at (DateTime nullable). Keep existing `is_active` column for backward compat during migration.
