    3. Compile: compile_workflow_to_stategraph(definition_json, user_context) — still MemorySaver in this plan

    6. In _run_graph: async for event in compiled.astream_events(...):
       - node_started → publish {event: "node_started", node_id}
       - node_completed → publish {event: "node_completed", node_id, output}
       - catch GraphInterrupt → publish {event: "hitl_paused", node_id, message}; set status=paused_hitl; return
    7. On completion: WorkflowRun.status = "completed"; store result_json
    8. On unhandled error: WorkflowRun.status = "failed"

    Redis publish: redis.Redis(url=settings.redis_url).publish(f"workflow:events:{run_id}", json.dumps(event_dict))

    Tests use mocked compile_workflow_to_stategraph, mocked DB session, mocked Redis.
