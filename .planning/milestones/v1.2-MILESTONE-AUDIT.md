### E2E Flows Verified

1. **Local user sign-in** (AUTH-04): /login → NextAuth Credentials → POST /api/auth/local/token → HS256 JWT → /chat
2. **Dual-issuer JWT dispatch** (AUTH-05): validate_token() peeks iss → routes HS256 local or RS256 Keycloak → identical UserContext
3. **Admin manages local users/groups** (AUTH-01, AUTH-02, AUTH-03): /admin/users → 8 proxy routes → 13 backend endpoints → ORM CRUD
4. **Prompt externalization** (INFRA-03, INFRA-04): master_agent.py + artifact_builder_prompts.py → load_prompt() → backend/prompts/*.md
5. **Artifact creation wizard** (ADMIN-02 through ADMIN-06): /admin/create → CopilotKit → fill_form tool → co-agent state → form pre-fill
6. **system.capabilities** (ECO-01): "what can you do" → keyword classify → capabilities_node → CapabilitiesCard A2UI
7. **OpenAPI bridge** (ECO-02): parse spec → select endpoints → register → tool_registry → /api/tools/call openapi_proxy dispatch
8. **Skill repository management** (ECO-03, ECO-04, ECO-05): add repo → browse → import → pending_review
9. **Skill export** (ECO-06): Export button → fetch zip → arrayBuffer proxy → StreamingResponse → download
10. **Admin credentials** (ADMIN-01): /admin/credentials → list all users' OAuth → revoke with optimistic UI

### Integration Risks (Informational)

1. **LOCAL_JWT_SECRET** must be in `backend/.env` — absence causes silent login failure (shows "Invalid credentials" instead of 500)
2. **Sub-agent prompt files** are pre-provisioned but disconnected — must add `load_prompt()` calls when sub-agents graduate from mock state
3. **intent_classifier.md** referenced in `prompts.py` docstring is stale — `FileNotFoundError` if example is copy-pasted
