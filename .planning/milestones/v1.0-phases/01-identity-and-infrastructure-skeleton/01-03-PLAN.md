RBAC behavior (has_permission):

employee role permissions:
- input: UserContext with roles=["employee"], permission="chat"
- output: True
- input: UserContext with roles=["employee"], permission="tool:admin"
- output: False

executive role permissions (read-only: chat + tool:reports only):
- input: UserContext with roles=["executive"], permission="chat"
- output: True
- input: UserContext with roles=["executive"], permission="tool:email"
- output: False

Multi-role union:
- input: UserContext with roles=["employee", "manager"], permission="workflow:create"
- output: True (manager has this, employee does not)

it-admin has all permissions:
- input: UserContext with roles=["it-admin"], permission="sandbox:execute"
- output: True

Team-lead inherits manager+employee:
- input: UserContext with roles=["team-lead"], permission="workflow:approve"
- output: True

Tool ACL behavior (check_tool_acl):

No ACL row = default allow:
- input: user_id=X, tool="email.fetch", no row in tool_acl for (X, "email.fetch")
- output: True (permit by default)

ACL row allowed=False:
- input: user_id=X, tool="email.fetch", row exists with allowed=False
- output: False

ACL row allowed=True:
- input: user_id=X, tool="email.fetch", row exists with allowed=True
- output: True

403 response format (per CONTEXT.md decisions):
- Must include: permission_required, user_roles, hint ("Contact IT admin")
- Example: {"detail": "Permission denied", "permission_required": "tool:email", "user_roles": ["executive"], "hint": "Contact IT admin"}

Audit log format:
- event: "tool_call"
- fields: user_id (str UUID), tool (str), allowed (bool), duration_ms (int)
- NEVER log: access_token, refresh_token, password, credentials

```python
# backend/security/rbac.py
from core.models.user import UserContext

ROLE_PERMISSIONS: dict[str, set[str]] = {
    "employee": {"chat", "tool:email", "tool:calendar", "tool:project"},
    "manager": {"chat", "tool:email", "tool:calendar", "tool:project", "tool:reports", "workflow:create"},
    "team-lead": {"chat", "tool:email", "tool:calendar", "tool:project", "tool:reports", "workflow:create", "workflow:approve"},
    "it-admin": {"chat", "tool:email", "tool:calendar", "tool:project", "tool:reports", "workflow:create", "workflow:approve", "tool:admin", "sandbox:execute", "registry:manage"},
    "executive": {"chat", "tool:reports"},
}


def get_permissions(roles: list[str]) -> set[str]:
    """Union of all permissions for a list of roles."""
    permissions: set[str] = set()
    for role in roles:
        permissions |= ROLE_PERMISSIONS.get(role, set())
    return permissions


def has_permission(user_context: UserContext, permission: str) -> bool:
    """Returns True if user has the required permission via any of their roles."""
    return permission in get_permissions(user_context["roles"])
```

**Tool ACL check and audit logging**:

```python
# backend/security/acl.py
import time
from uuid import UUID
from sqlalchemy import select
from sqlalchemy.ext.asyncio import AsyncSession
from core.models.tool_acl import ToolAcl
from core.logging import get_audit_logger
import structlog

logger = structlog.get_logger(__name__)
audit_logger = get_audit_logger()


async def check_tool_acl(
    user_id: UUID,
    tool_name: str,
    session: AsyncSession,
) -> bool:
    """
    Gate 3: Check per-user tool ACL.
    Returns True if allowed, False if explicitly denied.
    Default: allow when no row exists.
    user_id MUST come from JWT (get_current_user), never from request body.
    """
    result = await session.execute(
        select(ToolAcl).where(
            ToolAcl.user_id == user_id,
            ToolAcl.tool_name == tool_name,
        )
    )
    row = result.scalar_one_or_none()
    if row is None:
        return True  # default allow
    return row.allowed


async def log_tool_call(
    user_id: UUID,
    tool_name: str,
    allowed: bool,
    duration_ms: int,
) -> None:
    """Audit log every tool call attempt. Never log credentials."""
    audit_logger.info(
        "tool_call",
        user_id=str(user_id),
        tool=tool_name,
        allowed=allowed,
        duration_ms=duration_ms,
    )
```
