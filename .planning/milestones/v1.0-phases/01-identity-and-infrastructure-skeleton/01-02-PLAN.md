  <behavior>
JWT validation must cover ALL these cases:

Valid token → returns UserContext:
- input: valid RS256 JWT signed with Keycloak private key, correct iss, aud, not expired
- output: UserContext with user_id=UUID from sub, email, username from preferred_username, roles from realm_access.roles, groups from groups claim

Missing Authorization header → 401:
- input: request with no Authorization header
- output: HTTPException(status_code=401, detail="Not authenticated")

Expired token → 401:
- input: JWT with exp in the past
- output: HTTPException(status_code=401, detail="Token has expired")

Invalid signature → 401:
- input: JWT with tampered payload or wrong signing key
- output: HTTPException(status_code=401)

Wrong issuer → 401:
- input: JWT with iss != settings.keycloak_issuer
- output: HTTPException(status_code=401)

Wrong audience → 401:
- input: JWT with aud that does not include settings.keycloak_client_id
- output: HTTPException(status_code=401)

JWKS caching:
- input: 3 consecutive calls to validate_token with same kid
- output: only 1 HTTP call to Keycloak JWKS endpoint (cache hit for 2nd and 3rd)

Note on test strategy: Tests use mock JWKS + self-signed RSA keys. Do NOT call the real keycloak.blitz.local during unit tests (but integration test at the end of Phase 1 will). Use `unittest.mock.patch` or `pytest-mock` to mock httpx calls.

**JWKSCache** — in-process cache with TTL to avoid fetching on every request:

```python
# backend/security/jwt.py
import time
from uuid import UUID
from typing import Any
import httpx
from jose import jwt as jose_jwt, JWTError, ExpiredSignatureError
from core.config import settings
from core.models.user import UserContext
import structlog

logger = structlog.get_logger(__name__)

_JWKS_CACHE: dict[str, Any] = {}
_JWKS_CACHE_TTL = 300  # 5 minutes
_jwks_fetched_at: float = 0.0


async def _get_jwks() -> dict[str, Any]:
    global _jwks_fetched_at, _JWKS_CACHE
    now = time.monotonic()
    if _JWKS_CACHE and (now - _jwks_fetched_at) < _JWKS_CACHE_TTL:
        return _JWKS_CACHE
    async with httpx.AsyncClient() as client:
        resp = await client.get(settings.keycloak_jwks_url, timeout=10.0)
        resp.raise_for_status()
        _JWKS_CACHE = resp.json()
        _jwks_fetched_at = now
    return _JWKS_CACHE


async def validate_token(token: str) -> UserContext:
    """Validate RS256 JWT against Keycloak JWKS. Returns UserContext or raises HTTPException."""
    from fastapi import HTTPException
    try:
        jwks = await _get_jwks()
        payload = jose_jwt.decode(
            token,
            jwks,
            algorithms=["RS256"],
            audience=settings.keycloak_client_id,
            issuer=settings.keycloak_issuer,
        )
    except ExpiredSignatureError:
        raise HTTPException(status_code=401, detail="Token has expired")
    except JWTError as e:
        logger.warning("jwt_validation_failed", error=str(e))
        raise HTTPException(status_code=401, detail="Invalid token")

    roles: list[str] = payload.get("realm_access", {}).get("roles", [])
    return UserContext(
        user_id=UUID(payload["sub"]),
        email=payload.get("email", ""),
        username=payload.get("preferred_username", ""),
        roles=roles,
        groups=payload.get("groups", []),
    )
```

**FastAPI dependency**:

```python
# backend/security/deps.py
from fastapi import Depends, HTTPException, status
from fastapi.security import HTTPAuthorizationCredentials, HTTPBearer
from core.models.user import UserContext
from security.jwt import validate_token

_bearer = HTTPBearer(auto_error=False)


async def get_current_user(
    credentials: HTTPAuthorizationCredentials | None = Depends(_bearer),
) -> UserContext:
    if credentials is None:
        raise HTTPException(status_code=401, detail="Not authenticated")
    return await validate_token(credentials.credentials)
```
